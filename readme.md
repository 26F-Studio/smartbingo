# 聪明宾果游戏

每天一个小题目，做得出来说明你聪明，做不出来获得一天智障debuff，格林威治时间零点刷新

灵感来自qq群的“聪明宾果游戏”图片，不知道真正的原作者是谁所以没法署名了

注意相比原版调整了红色和蓝色的规则，“周围”只包括周围一圈八格，不包括自己

除了原版内容，还加入了更多的颜色以及最多/最少勾数挑战
